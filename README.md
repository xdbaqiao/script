Interesting shell scripts.
=======
**face_zsh.sh :    Change the zsh prompt**    
__________________
        
You can select face expression from  "\>\_<", "^_^", "T\_T", "$_$", "(￣﹁￣)", "^(oo)^" randomly.                      
It's so funny!          
    
**Result:**     
       
       yexinjing@vipsl ~ ^_^         
       yexinjing@vipsl ~ T_T        
       yexinjing@vipsl ~ >_<         
       yexinjing@vipsl ~ ^(oo)^           
       yexinjing@vipsl ~ (￣﹁￣)       
       yexinjing@vipsl ~ $_$           


**ip_ping.sh :    find the useful ip-address in LAN**    
__________________
      
You can get all ip-addresses in use.      
Need root.
