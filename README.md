Interesting shell scripts.
=======
**Change the zsh prompt**    
        
You can select face expression from  "\>\_<", "^_^", "T\_T", "$_$", "(￣﹁￣)", "^(oo)^" randomly.                      
It's so funny!          
    
**Result:**     
       
       yexinjing@vipsl ~ ^_^         
       yexinjing@vipsl ~ T_T        
       yexinjing@vipsl ~ >_<         
       yexinjing@vipsl ~ ^(oo)^           
       yexinjing@vipsl ~ (￣﹁￣)       
       yexinjing@vipsl ~ $_$
